# GivingHand Project 

## 📌 Overview

GivingHand is a community-driven charity application that enables users to create campaigns, donate resources, and track contributions using a modular Java EE architecture.

## 🎯 Phase 1 Scope

Phase 1 focuses on implementing core backend functionalities, including:

* User management (registration, login, profile update)
* Campaign creation and browsing
* Donation commitment and tracking
* Warehouse and inventory management

## 🎯 Features

* Role-based access (Donor / Organization)
* Campaign and donation handling
* Resource allocation with transaction integrity

## 🏗️ Structure

The system follows a layered architecture:

* **Controller Layer (JAX-RS):** Handles API requests
* **Business Layer (EJB):** Contains core logic
* **Persistence Layer (JPA):** Manages database operations

## ▶️ Running the Project

Deploy on a Java EE server (e.g., GlassFish or WildFly) and test APIs using Postman.

## 🎯 Objective

Build a clean and efficient backend system for managing campaigns and donations.
